# Jenny Speelman — Portfolio

Personal portfolio site, hosted on GitHub Pages.

## Deploy to GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to **Deploy from a branch** → `main` → `/ (root)`
4. Your site will be live at your GitHub Pages URL

## Local preview

```bash
npx serve .
```

## Updating your photo

Replace `images/jenny.jpg` — keep the filename or update the `src` in `index.html`.
